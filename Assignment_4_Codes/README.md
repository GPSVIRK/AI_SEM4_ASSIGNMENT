# AI Assignment

This repository contains solutions for multiple assignment questions organized into separate folders.

## ▶️ How to Run

### Q1 & Q2 (Map Colouring)

```bash
cd Q1_and_Q2
g++ mapColouring.cpp -o mapColouring
./mapColouring
```

* Runs the C++ solver
* Automatically calls the Python visualizer
* Generates output SVG files

---

### Q3 (Sudoku Solver)

```bash
cd Q3
python Q3.py
```

* Runs multiple test cases sequentially
* Press Enter to navigate through solutions

---

### Q4 (Cryptarithm Solver)

```bash
cd Q4
python Q4.py
```

* Outputs all valid solutions for the given puzzle

---
